About Women's Football Scores

Learn about this project and its purpose

Project Overview

This site was created as an IS117 project to demonstrate responsive web design, Bootstrap layout, and data presentation for women's football leagues worldwide.

The project showcases five major women's football competitions:

  • UEFA Women's Champions League
  • Liga F (Spain)
  • Women's Super League (England)
  • Liga MX Femenil (Mexico)
  • National Women's Soccer League (United States)

The content and page structure were developed from project assignment guidelines, focusing on creating an accessible, responsive, and visually appealing sports information website.

Website Features
Design & Layout
  • Responsive Bootstrap 5.3.2 framework
  • Dark theme with accessible color contrast
  • Card-based layout for content presentation
  • Different homepage and interior page templates
  • Gradient navigation bar
Functionality
  • Live match scores and upcoming fixtures
  • League standings tables
  • Top scorers tracking
  • Multi-level navigation dropdown
  • Contact and subscription forms

Educational Purpose

This website is an educational project created for IS117 - Introduction to Website Development at New Jersey Institute of Technology (NJIT).

Learning Objectives Demonstrated:

  • Semantic HTML structure and accessibility
  • Responsive web design principles
  • Bootstrap framework implementation
  • CSS custom styling and theming
  • JavaScript for interactive elements
  • Git and GitHub workflow
  • Multi-page website architecture
